# Natural Language SQL Assistant

## Overview

An AI-powered application that converts natural language questions into SQL queries and retrieves database results automatically.

## Features

* Natural language to SQL conversion
* FastAPI backend
* Gemini API integration
* SQLite database execution
* AI-generated explanations
* Swagger UI testing

## Tech Stack

* Python
* FastAPI
* LangChain
* Gemini API
* SQLite
* SQLAlchemy

## Example

Input:
Show all employees

Generated SQL:
SELECT * FROM employees;

Output:
Returns employee records and natural language explanation.
